O que é CSS?

Introdução ao CSS

CSS (Cascading Style Sheets) é uma linguagem de estilo utilizada para descrever a apresentação de um documento escrito em HTML ou XML. Enquanto o HTML é responsável pela estrutura da página, o CSS é responsável por definir como os elementos dessa estrutura irão aparecer no navegador. Ou seja, o CSS trata da aparência visual do site, como cores, fontes, tamanhos e espaçamento.

O que o CSS faz?

O CSS permite que você:

  • Defina cores de fundo e texto;
  • Escolha fontes e tamanhos de texto;
  • Controle o layout da página, como o posicionamento de elementos;
  • Adicione animações e transições;
  • Crie responsividade, ou seja, ajuste o design para diferentes tamanhos de tela, como dispositivos móveis e desktops;
  • E muito mais!

Exemplo de como usar CSS

O código CSS pode ser escrito dentro de uma tag <style> no cabeçalho da página ou em um arquivo externo. Veja um exemplo básico de CSS:

<style>
    body {
        background-color: #f0f0f0;
        font-family: Arial, sans-serif;
    }

    h1 {
        color: #4CAF50;
    }

    p {
        color: #333;
        font-size: 16px;
    }
</style>

Conclusão

O CSS é uma parte essencial do desenvolvimento web moderno. Ele permite que os desenvolvedores e designers criem sites visualmente atraentes e funcionais, separando a estrutura do conteúdo e a apresentação da página. Com o CSS, é possível transformar um site simples em uma experiência de usuário rica e personalizada.
